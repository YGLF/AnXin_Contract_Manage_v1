# 安信合同管理系统 - 数据库设计文档

本文档详细描述系统的数据库表结构和关系。

## 1. 数据库概述

- 数据库名称：contract_manage
- 字符集：utf8mb4
- 排序规则：utf8mb4_unicode_ci
- 存储引擎：InnoDB

---

## 2. 表结构

### 2.1 用户表 (users)

存储系统用户信息。

| 字段名 | 类型 | 可空 | 默认值 | 说明 |
|--------|------|------|--------|------|
| id | uint | 否 | 自增 | 用户ID，主键 |
| username | varchar(50) | 否 | - | 用户名，唯一 |
| email | varchar(100) | 是 | - | 邮箱 |
| hashed_password | varchar(200) | 否 | - | bcrypt加密密码 |
| password_hash | varchar(64) | 是 | - | SHA-256杂凑值 |
| hash_verified | bool | 是 | false | 杂凑验证是否通过 |
| integrity_hash | varchar(64) | 是 | - | 数据完整性哈希 |
| full_name | varchar(100) | 是 | - | 真实姓名 |
| role | varchar(20) | 是 | user | 用户角色 |
| custom_permissions | text | 是 | - | 自定义权限(JSON) |
| department | varchar(100) | 是 | - | 所属部门 |
| phone | varchar(20) | 是 | - | 联系电话 |
| is_active | bool | 是 | true | 是否激活 |
| account_status | varchar(20) | 是 | permanent | 账号状态 |
| valid_from | datetime | 是 | - | 有效期开始 |
| valid_to | datetime | 是 | - | 有效期结束 |
| valid_hours | int | 是 | 0 | 临时账号有效期小时数 |
| created_at | datetime | 否 | 当前时间 | 创建时间 |
| updated_at | datetime | 是 | - | 更新时间 |

**索引**：
- PRIMARY KEY (id)
- UNIQUE INDEX idx_username (username)
- INDEX idx_email (email)
- INDEX idx_role (role)

### 2.4 合同表 (contracts)

存储合同核心信息。

| 字段名 | 类型 | 可空 | 默认值 | 说明 |
|--------|------|------|--------|------|
| id | uint | 否 | 自增 | 合同ID，主键 |
| contract_no | varchar(50) | 否 | - | 合同编号，唯一 |
| title | varchar(200) | 否 | - | 合同标题 |
| customer_id | uint | 是 | - | 关联客户ID |
| contract_type_id | uint | 是 | - | 关联合同类型ID |
| amount | decimal(15,2) | 是 | 0 | 合同金额 |
| currency | varchar(10) | 是 | CNY | 货币单位 |
| status | varchar(20) | 是 | draft | 合同状态 |
| sign_date | datetime | 是 | - | 签订日期 |
| start_date | datetime | 是 | - | 开始日期 |
| end_date | datetime | 是 | - | 结束日期 |
| payment_terms | text | 是 | - | 付款条款 |
| content | text | 是 | - | 合同正文 |
| notes | text | 是 | - | 备注 |
| creator_id | uint | 是 | - | 创建人ID |
| created_at | datetime | 否 | 当前时间 | 创建时间 |
| updated_at | datetime | 是 | - | 更新时间 |

**索引**：
- PRIMARY KEY (id)
- UNIQUE INDEX idx_contract_no (contract_no)
- INDEX idx_title (title)
- INDEX idx_customer_id (customer_id)
- INDEX idx_contract_type_id (contract_type_id)
- INDEX idx_status (status)
- INDEX idx_creator_id (creator_id)

**合同状态说明**：
- draft - 草稿
- pending - 待审批
- approved - 已批准
- active - 已生效
- in_progress - 执行中
- pending_pay - 待付款
- completed - 已完成
- terminated - 已终止
- archived - 已归档

## 3. 表关系

```
users (1) ────────── (N) contracts
  │                          │
  └──────── (N) approval_records
  └──────── (N) contract_executions
  └──────── (N) notifications

customers (1) ───── (N) contracts

contract_types (1) ─ (N) contracts

contracts (1) ──── (N) approval_workflows
  │                    │
  └────── (N) contract_executions
  │                    │
  └────── (N) documents
  │                    │
  └────── (N) reminders
  │                    │
  └────── (N) notifications
  │                    │
  └────── (N) contract_lifecycle_events

approval_workflows (1) ─── (N) workflow_approvals
```

## 5. 安全配置

### 登录安全

- 最大登录尝试次数：5
- 锁定时长：15分钟
- 失败记录重置时间：30分钟

### 密码安全

- 密码加密：bcrypt
- 密码杂凑验证：SHA-256
