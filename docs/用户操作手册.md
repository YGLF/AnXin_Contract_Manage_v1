## 系统登录

### 登录流程

1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 点击登录按钮

### 登录说明

- 首次登录后建议修改密码
- 密码错误5次后账号将被锁定
- 30分钟无操作自动退出

### 忘记密码

请联系系统管理员重置密码。

## 客户管理

### 客户列表

位置：`客户管理` 菜单

功能：
- 查看所有客户
- 按类型筛选（客户/供应商）
- 搜索客户名称

### 创建客户

1. 点击 `新增客户` 按钮
2. 填写客户信息：
   - 名称（必填）
   - 代码（必填，唯一）
   - 类型：客户/供应商
   - 联系人
   - 电话
   - 邮箱
   - 地址
   - 信用等级
3. 点击保存

### 编辑客户

1. 在客户列表点击客户名称
2. 点击 `编辑` 按钮
3. 修改信息后保存

### 删除客户

1. 选中客户记录
2. 点击 `删除` 按钮
3. 确认删除操作

> 注意：有关联合同的客户不能删除

## 合同管理

### 合同列表

位置：`合同管理` 菜单

功能：
- 查看所有合同
- 按状态筛选
- 按客户筛选
- 搜索合同标题/编号

### 合同状态

| 状态 | 说明 |
|------|------|
| 草稿 | 合同正在编辑 |
| 待审批 | 已提交等待审批 |
| 已批准 | 审批通过等待生效 |
| 已生效 | 合同已开始执行 |
| 执行中 | 合同正在执行 |
| 待付款 | 等待付款 |
| 已完成 | 合同执行完毕 |
| 已终止 | 合同提前终止 |
| 已归档 | 合同已归档保存 |

### 创建合同

1. 点击 `新增合同` 按钮
2. 填写合同信息：
   - 标题（必填）
   - 客户（必填）
   - 合同类型（必填）
   - 金额
   - 货币
   - 签订日期
   - 开始日期
   - 结束日期
   - 付款条款
   - 合同内容
   - 备注
3. 点击保存（状态为草稿）

### 提交审批

1. 在合同列表点击合同标题
2. 点击 `提交审批` 按钮
3. 确认提交

提交后将进入审批流程，状态变为 `待审批`。

### 合同详情

点击合同标题进入详情页，包含：

- 基本信息
- 执行跟踪
- 文档管理
- 审批记录
- 生命周期

### 状态变更

部分状态变更需要审批：
- 归档
- 终止
- 执行中
- 待付款

申请状态变更：
1. 点击 `申请状态变更` 按钮
2. 选择目标状态
3. 填写原因
4. 提交申请

## 审批管理

### 待审批列表

位置：`审批管理` 菜单

显示当前用户需要审批的合同列表。

### 审批操作

1. 点击合同标题查看详情
2. 在审批区域：
   - 选择审批结果：同意/拒绝
   - 填写审批意见
3. 点击提交

### 审批流程

三级审批流程：
1. 销售总监审批
2. 技术总监审批
3. 财务总监审批

上一级审批通过后，才会流转到下一级。

### 审批记录

在合同详情页的 `审批记录` 标签查看历史审批记录。

## 到期提醒

### 提醒列表

位置：`到期提醒` 菜单

显示所有设置了提醒的合同。

### 创建提醒

1. 在合同详情页点击 `添加提醒` 按钮
2. 设置提醒日期
3. 设置提前提醒天数
4. 点击保存

### 查看提醒

- 到期前7天、30天自动显示在仪表盘
- 提醒到期时发送通知

## 用户管理

> 仅管理员可见

### 用户列表

显示所有系统用户。

### 创建用户

1. 点击 `新增用户` 按钮
2. 填写用户信息：
   - 用户名（必填）
   - 密码（必填）
   - 邮箱
   - 姓名
   - 角色
   - 部门
   - 电话
3. 点击保存

### 角色说明

| 角色 | 说明 |
|------|------|
| 超级管理员 | 系统全部权限 |
| 销售总监 | 审批销售提交的合同 |
| 技术总监 | 审批技术相关合同 |
| 财务总监 | 审批财务相关合同 |
| 合同管理员 | 可查看所有合同 |
| 销售人员 | 只能查看自己的合同 |
| 审计管理员 | 查看审计日志 |

### 权限配置

在用户编辑页面，可以为用户追加自定义权限。

## 常见问题

### 1. 无法创建合同

检查：
- 是否选择了客户和合同类型
- 必填字段是否填写完整

### 2. 无法提交审批

检查：
- 合同状态是否为草稿
- 客户和合同类型是否已填写

### 3. 审批被拒绝怎么办

查看审批意见，修改合同后重新提交审批。

### 4. 收不到通知提醒

检查：
- 是否开启浏览器通知
- 通知是否被拦截

### 5. 文件上传失败

检查：
- 文件大小是否超过限制（10MB）
- 文件格式是否支持
